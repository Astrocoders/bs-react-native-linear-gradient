# bs-react-native-linear-gradient

Bucklescript bindings for react-native-linear-gradient https://github.com/react-native-community/react-native-linear-gradient/

# Demo

```ml
      <LinearGradient
        colors=[|
          "rgba(35, 38, 66, 0)",
          "rgba(35, 38, 66, .1)",
          "rgba(35, 38, 66, .2)",
        |]
        locations=[|0.7, 0.75, 1.|]>
         children
      </LinearGradient>
```
